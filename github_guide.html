GitHub × Claude Code 사전 준비 안내

동아리 활동 전에 미리 설치하고 가입해 두실 내용을 정리했습니다

이번 동아리 활동에서 함께 해볼 일

  • 공동 레포지토리(저장소)를 만들어 함께 코드를 관리합니다.
  • GitHub Pages / Cloudflare Pages로 웹사이트를 직접 배포(deploy)해 봅니다.
  • Claude Code로 AI에게 작업을 시키고 결과를 깃허브에 올리는 흐름을 익힙니다.

당일에는 실습 시간이 빠듯합니다. 아래 1~3번까지는 꼭 미리 마치고 와 주세요.

1기본 프로그램 설치

GitHub를 쓰려면 컴퓨터에 두 가지 도구가 필요합니다. Git(버전 관리 도구)과 VS Code(코드 편집기)입니다.

1-1. Git 설치

Git은 "파일이 언제, 누가, 어떻게 바뀌었는지" 기록하는 프로그램입니다. GitHub와 통신하려면 반드시 필요합니다.

Windows

  1. git-scm.com 접속 → 자동 다운로드
  2. 설치 옵션은 모두 기본값으로 두고 Next 진행
  3. 설치 후 Git Bash가 시작 메뉴에 생기면 성공
다운로드

macOS

  1. 터미널 열기 (Spotlight → "터미널")
  2. xcode-select --install 입력
  3. 설치 안내창이 뜨면 "설치" 클릭

이미 Xcode가 깔려 있다면 그대로 사용 가능합니다.

설치 확인 방법: 터미널(또는 Git Bash)에서 git --version을 입력해 git version 2.xx.x처럼 나오면 OK입니다.

1-2. VS Code(Visual Studio Code) 설치

코드를 작성하고 Git/GitHub과 연동하는 가장 일반적인 편집기입니다. Claude Code도 이 안에서 사용합니다.

  1. code.visualstudio.com 접속
  2. 본인 운영체제(OS)에 맞는 버전 다운로드 → 설치
  3. 설치 후 실행해 메뉴가 한글로 보이지 않는다면, 왼쪽 사이드바 ⚙ → Extensions에서 "Korean Language Pack"을 설치하면 한글로 바뀝니다.
VS Code 다운로드

1-3. Node.js 설치 (선택 사항)

예전에는 Claude Code를 쓰려면 Node.js 설치가 꼭 필요했지만, 지금은 VS Code 확장 프로그램만 깔면 자동으로 모두 해결됩니다. 그래서 이 항목은 건너뛰셔도 무방합니다.

개발에 익숙해지신 후 자체 도구를 설치하고 싶으실 때 nodejs.org에서 LTS 버전을 받아두시면 됩니다.

참고 — 학교/교육청 컴퓨터에서 설치가 막힐 때: 관리자 권한이 필요한 경우, 본인 노트북을 가져오시거나 미리 정보부서에 설치 요청을 해 두시는 편이 좋습니다.

2AI 활용을 위한 사전 지식

Claude Code 같은 AI 코딩 도구를 활용하려면 몇 가지 용어와 개념을 알고 계시면 훨씬 수월합니다. 외울 필요는 없고, "이런 게 있구나" 정도로 훑어보시면 됩니다.

2-1. 꼭 알아둘 핵심 용어

Repository (레포, 저장소)
프로젝트 파일들이 담긴 폴더. GitHub의 가장 기본 단위입니다.
Commit (커밋)
"이 시점의 상태를 저장!"이라는 도장. 변경 이력을 남기는 단위.
Push / Pull
내 컴퓨터의 변경을 GitHub에 올리는 게 push, 반대로 받아오는 게 pull.
Branch (브랜치)
본 줄기에서 가지를 쳐서 따로 작업하는 공간. 협업의 핵심.
Deploy (배포)
코드를 실제 웹사이트 등으로 띄워 누구나 볼 수 있게 하는 일.
Terminal / CLI
검은 화면에 명령어로 컴퓨터를 조작하는 도구. Git Bash, 터미널 등.

2-2. Claude Code란?

Claude Code는 Anthropic이 만든 AI 코딩 도구입니다. "내 컴퓨터의 폴더와 파일을 직접 보고 수정해 주는 AI"라고 이해하시면 쉽습니다.

  • 채팅창에 코드를 복사·붙여넣기 하지 않아도 됩니다 — AI가 직접 파일을 읽고 고칩니다.
  • "이 폴더 안에서 HTML 파일을 만들고 배포 준비해 줘" 같은 자연어 지시가 가능합니다.
  • 한국어로 입력하면 한국어로 답합니다 — 별도 설정 불필요.
  • 설치는 VS Code 확장 프로그램 하나면 끝입니다. 터미널 명령이나 Node.js 설치가 필요 없어졌습니다. 동아리 시간에 함께 설치해 보니, 지금 미리 설치하실 필요는 없습니다.
참고 — Claude 계정 안내: Claude Code는 Claude Pro 이상 유료 계정이 필요합니다(월 $20부터). 동아리 시간에는 강사/운영자 계정으로 시연 위주로 진행할 예정이니, 미리 결제하실 필요는 없습니다. 본인 업무에 본격 활용해 보고 싶으신 분만 시범 사용 후 결정하시면 됩니다.

2-3. AI에게 일을 시킬 때 도움이 되는 마음가짐

① 한 번에 큰 덩어리를 던지지 말고, 작게 쪼개기
"웹사이트 만들어 줘"보다 "한 페이지짜리 HTML로 자기소개 페이지, 제목과 사진 자리, 연락처 섹션 포함" 처럼 구체적으로.
② 결과가 어색하면 "왜 그렇게 했어?" 묻기
AI도 실수합니다. 그럴 땐 다시 시키지 말고, 의도를 설명해 달라고 한 뒤 함께 수정 방향을 잡으시면 좋습니다.
③ 만든 결과는 반드시 한 번 직접 확인하기
AI가 "다 됐어요"라고 해도, 브라우저에서 열어보거나 직접 실행해 보시는 습관이 중요합니다.
잠깐! AI 도구가 만든 코드를 깃허브에 올릴 때는, 비밀번호·API 키·개인정보가 코드 안에 들어가 있지 않은지 한 번 확인하는 습관을 들이세요. 공개 레포에 올라가면 누구나 볼 수 있습니다.

3GitHub 가입하기

GitHub는 무료로 가입할 수 있습니다. 평소에 자주 쓰시는 이메일로 가입하시면 됩니다.

가입 절차

  1. github.com/signup 접속
  2. 이메일 입력 → Continue
    ※ 업무용보다는 개인 이메일(Gmail 등)을 권장합니다. 인사이동·퇴직 후에도 계정이 유지됩니다.
  3. 비밀번호 설정 (15자 이상, 또는 8자 이상 + 숫자·소문자 포함)
  4. 사용자명(Username) 설정
    ※ 다른 사람과 겹치지 않는 영문 ID. 예: kim-sihyung, cne-teacher01 등. 한 번 정하면 자주 바꾸기 어려우니 신중하게.
  5. 이메일 수신 동의 여부 선택 → Continue
  6. 퍼즐로 사람인지 확인 → 이메일로 받은 인증 코드 입력
  7. 설문(선택사항)은 모두 건너뛰셔도 됩니다 → Free 플랜 선택
가입 완료 확인: 로그인 후 우측 상단에 본인의 프로필 사진(또는 기본 아이콘)이 보이고, 좌측에 "Create your first repository"가 나오면 정상입니다.

가입 후 권장 작업 (5분)

  1. 프로필 사진 설정 — 우측 상단 아바타 → Settings → Profile → Picture
    동아리에서 누가 누구인지 알아보기 좋게 본인 사진이나 캐릭터를 올려두세요.
  2. 표시 이름(Name) 한글로 설정 — 같은 페이지에서 Name 칸에 "김시형" 등 표시할 이름 입력
  3. 2단계 인증(2FA) 설정 권장 — Settings → Password and authentication → Two-factor authentication
    의무는 아니지만 보안상 권장됩니다. 스마트폰 인증 앱(Google Authenticator 등)으로 설정 가능.

✓출발 전 체크리스트

동아리 활동 당일 아침, 아래 항목들을 한 번씩 확인해 주세요. 모두 ✅이면 곧바로 실습에 들어갈 수 있습니다.

  • Git 설치 완료 (git --version 실행 시 버전 표시됨)
  • VS Code 설치 완료 및 실행 가능
  • GitHub 계정 가입 및 이메일 인증 완료
  • GitHub에 로그인 가능 (브라우저 즐겨찾기에 추가해 두면 편함)
  • 본인이 가입할 때 쓴 GitHub 사용자명과 이메일을 메모해 둠
  • 노트북 전원 어댑터 지참, 와이파이 접속 확인
설치 중 막히는 부분이 있으시면 동아리 단톡방 또는 담당자에게 미리 알려 주세요. 당일 설치부터 시작하면 실습 시간이 부족합니다.

?자주 묻는 질문

Q. 코딩을 한 번도 안 해봤는데 따라갈 수 있을까요?
네, 충분합니다. 이번 동아리는 "코드를 직접 짜는 법"이 아니라 "AI에게 시키고, 결과를 깃허브로 관리·배포하는 법"을 익히는 자리입니다. 기본 마우스 조작과 한글 타자만 되시면 OK입니다.
Q. 학교 컴퓨터에서 프로그램 설치가 안 됩니다.
관리자 권한이 막혀 있을 가능성이 큽니다. 가능하면 개인 노트북을 가져오시거나, 정보부서에 미리 협조 요청을 해 두세요.
Q. Claude Code는 유료인가요?
네, Claude Pro 이상 유료 계정(월 $20부터)이 필요합니다. 다만 동아리 시간에는 운영자 계정으로 시연 위주로 진행할 예정이니, 미리 결제하실 필요는 없습니다. 직접 사용해 보고 업무에 도움이 되겠다 싶을 때 가입을 고려해 보세요.
Q. 깃허브에 올린 내용을 나중에 지울 수 있나요?
네, 본인이 만든 레포는 언제든 삭제할 수 있습니다. 다만 다른 사람이 이미 복제(fork/clone)해 갔다면 그 사본은 사라지지 않습니다. 그래서 비밀번호·개인정보는 처음부터 올리지 않는 것이 중요합니다.
